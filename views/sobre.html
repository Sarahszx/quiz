Objetivo do projeto

Este jogo foi desenvolvido com o objetivo de ensinar e reforçar conceitos de lógica de programação de forma interativa e divertida. Voltado para iniciantes, estudantes e entusiastas da área de tecnologia, ele oferece desafios progressivos que estimulam o raciocínio lógico, a resolução de problemas e a construção de algoritmos básicos.

Este projeto é uma iniciativa orientada pelo professor Fábio Fraga.
